### Overview

An interactive web app built with **Streamlit** to predict house sale prices using a **Random Forest Regressor** trained on the [Kaggle House Prices Dataset](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques).

### Features

- User input via sidebar (e.g., quality, living area, garage, etc.)
- Real-time price prediction with confidence interval
- Feature importance chart and SHAP explanation
- Downloadable predictions as CSV
